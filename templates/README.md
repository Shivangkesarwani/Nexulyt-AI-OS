# Templates Directory Root

This directory contains production-ready system architecture, component directory, API interface, and database modeling templates for various application archetypes.

---

## Overview

Templates serve as structured specifications to bootstrap software development. They define recommended stack matrices, folder layouts, database normalization expectations, API endpoints structures, security vectors, and performance bounds for specific application types (e.g. SaaS, E-commerce, LMS).

These templates integrate with the `Full-Stack Orchestrator` to determine how to activate and guide specialist skills.
